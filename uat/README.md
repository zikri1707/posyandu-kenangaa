## Panduan Mengubah File Markdown (.md) ke PDF / Word

Jika Anda membutuhkan dokumen ini dalam format PDF atau Microsoft Word untuk laporan/tugas UAS:
1.  **Menggunakan VS Code**:
    *   Pasang ekstensi **Markdown PDF** atau **Markdown All in One** di VS Code.
    *   Buka file `.md`, klik kanan di dalam editor, lalu pilih **Markdown PDF: Export (pdf)**.
2.  **Menggunakan Obsidian / Typora**:
    *   Buka file `.md` menggunakan aplikasi Typora atau Obsidian, lalu gunakan fitur **Export to PDF** atau **Export to HTML**.
3.  **Menggunakan Konverter Online**:
    *   Anda dapat mengonversi file Markdown ini ke PDF secara langsung menggunakan situs seperti [dillinger.io](https://dillinger.io/) atau [pandoc](https://pandoc.org/).
