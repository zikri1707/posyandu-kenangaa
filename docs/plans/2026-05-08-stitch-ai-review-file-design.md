# Design Document: Stitch AI Dashboard Review File

## Overview
This document outlines the creation of a static HTML version of the Posyandu Admin Dashboard for AI UI/UX review.

## Requirements
- Must be a single `.html` file.
- Must use Tailwind CSS for styling.
- Must include all custom CSS classes (`hero-mesh`, `neo-glass`, etc.).
- Must contain mock data for statistics and tables.
- Must render charts using Chart.js.

## Component Extraction
- Hero Section: Extracted from `admin-dashboard.blade.php`.
- KPI Cards: 6 cards with mock numbers.
- Stunting Table: 3-5 rows of sample patient data.
- Widgets: Agenda, Immunization, and Nutrition Distribution charts.
